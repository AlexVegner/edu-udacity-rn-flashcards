## Installation instructions:

Setup app mobile-flashcards folder

1. Open
```
cd mobile-flashcards
```

2. Install node packages
```
npm install
```
or with yarn
```
yarn install
```

3. Start packager
```
npm start
```
or with yarn
```
yarn start
```

4. Start android app 'a', start ios app 'i'
